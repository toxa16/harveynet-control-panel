# harveynet-control-panel

HarveyNet Control Panel.

## Environment Variables

- REACT_APP_BASE_URL
- REACT_APP_OWNERSHIP_SERVER_URL

- REACT_APP_AUTH0_URL
- REACT_APP_AUTH0_CLIENT_ID

- REACT_APP_PUSHER_APP_KEY
- REACT_APP_PUSHER_CLUSTER (default: "eu")
- REACT_APP_PUSHER_AUTH_ENDPOINT
